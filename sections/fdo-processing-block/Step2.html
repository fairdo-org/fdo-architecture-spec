Description

This process shall resolve the identifier using the resolvers from the Iist of Identifier-Resolvers provided by Step 1.

Inputs

A list of Identifer-Resolvers from step 1

An optional flag TRY_ALL that will try all resolvers. The default is to stop at the first successful resolution.

Outputs

For each resolver specified in list of resolvers that succeed, the following information is returned.

  • The name of the resolver.
  • Resolution status: RESOLVED
  • The Identifier Record.

If the resolution is not successful:

  • The name of the resolver
  • Resolution status: NOT_RESOLVED
  • The resolvers error message if any.
Expected behavior

The process will try each resolver until one is successful or ,if specified, all successful resolutions. This process will return each resolver’s respective identifier record. The process MUST prevent false positives.
